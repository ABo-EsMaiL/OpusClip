# OpusClip Bundled Fonts

This directory contains the bundled TrueType (`.ttf`) fonts required by the pipeline for subtitle rendering.

**Zero-Download Policy Note**:
Per the OpusClip Constitution, the automatic downloading of font assets during pipeline execution or development is prohibited. You must manually place the required TTF files into this directory before running the pipeline in production.

Required fonts (SIL OFL licensed):
- `Tajawal-ExtraBold.ttf`
- `Tajawal-Bold.ttf`
- `Montserrat-ExtraBold.ttf`
- `Montserrat-Bold.ttf`
- `Amiri-Regular.ttf`
- `NotoSansArabic-Regular.ttf` (or subset)
